The Student Record Management System is a simple program made using C language. It helps users to store and manage student information easily. The program keeps details like
student name, roll number, marks for theory, practical, and viva, as well as attendance percentage. Based on the marks entered, the system automatically calculates total marks,
assigns grades, and decides whether the student has passed or failed. All data is saved using file handling, so that records are not lost when the program is closed. They are
managed using a standard CSV (Comma-Separated Values) file, allowing records to be accessed or edited by other spreadsheet software if needed.
